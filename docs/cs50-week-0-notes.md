# CS50x Week 0 Notes

## Main idea

Computer Science is about solving problems using inputs, algorithms, and outputs.

## Key concepts

### Binary
Computers represent information using 0s and 1s.

### Bit and byte
A bit is one 0 or 1.
A byte is 8 bits.

### Algorithm
An algorithm is a step-by-step process for solving a problem.

### Efficiency
Some algorithms solve problems faster than others.

### Pseudocode
Pseudocode is planning a program using normal language before writing real code.

### Functions
Functions are actions that perform tasks.

### Conditions
Conditions let programs make decisions.

### Loops
Loops repeat actions.

### Variables
Variables store information.

### Abstraction
Abstraction hides complicated details so we can focus on bigger ideas.
