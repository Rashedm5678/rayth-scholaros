# Rayth ScholarOS Project Plan

## Version 0

The first version of Rayth ScholarOS will be a simple AI study assistant.

## Core features

1. Notes summarizer
2. Flashcard generator
3. Quiz generator
4. Study planner

## Why this matters

Many students struggle with organizing notes and preparing for exams. Rayth ScholarOS will help students turn messy notes into clear study material.

## Future features

- PDF upload
- Local AI model support
- AI agent dashboard
- Weak-topic tracker
- Exam revision planner
